üí° How Pagination Works

ArcGIS automatically pages through data:

  • Only loads features in current map extent
  • Requests 100-1000 records per query
  • Uses resultOffset and resultRecordCount
  • Caches loaded features client-side

Try this:

  • Zoom in/out and watch queries execute
  • Pan around - only visible area loads
  • Open browser console (F12) to see details
